### Menu

| 属性        | 说明                 | 类型             | 默认值     |
| ----------- | -------------------- | ---------------- | ---------- |
| closable	      |是否显示左上角的关闭按钮		             | boolean |true    |
| mask  | 是否展示遮罩	         | boolean  | true      |
| maskClosable      |点击蒙层是否允许关闭	             | boolean  |true|
| placement  | 抽屉的方向	      | 'top'、 'right' 、 'bottom'、 'left' |   right    |
| open  | Drawer 是否可见   | boolean 	 |    -    |
| onClose  | 点击遮罩层或左上角叉或取消按钮的回调	              | function(e)	 |-        |
