### 何时使用

集合了组件库中所有的图标，通过 type 进行图标类型指定。

### API

| 属性  | 说明     | 类型          | 默认值 |
| ----- | -------- | ------------- | ------ |
| type  | 指定图标 | string        | -      |
| style | 样式     | CSSProperties | -      |
