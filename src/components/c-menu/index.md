### API

| 属性        | 说明                 | 类型             | 默认值     |
| ----------- | -------------------- | ---------------- | ---------- |
| mode  | 菜单类型，现在支持垂直 vertical、水平 horizontal         | string  `vertical`、`horizontal`            | `vertical`      |
| triggerSubMenuAction  | 如何触发菜单选中事件               | click、hover |horizontal下默认hover,vertical下默认click        |
| onClick  | 菜单点击事件               | - |-        |
| selectedKeys      |默认选中的菜单             | - | -         |
