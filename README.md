# dumi-demo

## Getting Started

Install dependencies,

```bash
$ npm i
```

Start the dev server,
```bash
yarn add cloud-react --save
```

### 框架依赖版本
- `react` >= `16.8.6`
- `react-dom` >= `16.8.6`
- `react-router-dom` >= `5.0.0`

### 支持浏览器
- `chrome` >= `56`
- `firefox` >= `59`

### 示例
```js
// 引入样式
import 'cloud-react/cloud-react.css';

```bash
$ npm start
```

Build documentation,

```bash
$ npm run docs:build
```

Build library via `father-build`,

```bash
$ npm run build
```
